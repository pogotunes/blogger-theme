A comprehensive SEO audit is the foundation of any successful search optimization strategy. This complete checklist covers 50+ critical checkpoints across technical SEO, on-page optimization, content, off-page factors, and user experience. Use this to identify what's working and what needs fixing on your website.

How to Use This Checklist

Work through each section systematically. Mark items as pass, fail, or not applicable. For failures, note the priority level (High/Medium/Low) and estimated effort. When you're done, you'll have a complete action plan for improving your search performance.

1. Technical SEO Foundation

Crawlability & Indexation

  • Search console is verified and error-free
  • XML sitemap is submitted and current
  • robots.txt is correctly configured
  • No important pages blocked from crawling
  • Crawl errors are minimal (4xx, 5xx, DNS, server)
  • Important pages are indexed (site:domain.com check)
  • Thin/low-value pages are noindexed
  • Pagination is handled correctly (rel next/prev)

Site Architecture & URLs

  • URL structure is clean and descriptive
  • HTTPS is enabled and enforced
  • No duplicate content issues
  • Canonical tags are correctly implemented
  • Important pages reachable within 3 clicks from homepage
  • Breadcrumb navigation is implemented
  • Internal linking structure is logical

Core Web Vitals & Performance

  • LCP (Largest Contentful Paint) under 2.5s
  • FID (First Input Delay) under 100ms
  • CLS (Cumulative Layout Shift) under 0.1
  • Mobile page speed score 90+
  • Desktop page speed score 90+
  • Images are optimized (WebP, compressed, lazy loaded)
  • JavaScript and CSS are minified
  • Render-blocking resources are minimized

Schema & Structured Data

  • Organization schema implemented
  • LocalBusiness schema (if applicable)
  • Article schema on blog posts
  • FAQ schema on FAQ pages
  • BreadcrumbList schema implemented
  • Product schema on product pages (if ecommerce)
  • No schema markup errors in Google Search Console
  • JSON-LD format used for structured data

2. On-Page SEO

Content & Keywords

  • Each page targets a unique primary keyword
  • Title tags are unique, descriptive, and under 60 chars
  • Meta descriptions are compelling and under 160 chars
  • H1 tag is present and includes primary keyword
  • Heading hierarchy is logical (H1 > H2 > H3)
  • Content is 1000+ words (or appropriate for intent)
  • Keyword is naturally included in first 100 words
  • Images have descriptive alt text
  • Internal links to related content exist
  • Outbound links to authoritative sources
  • Content matches search intent (informational/navigational/transactional)

3. User Experience

  • Website is fully responsive/mobile-friendly
  • Touch elements are properly sized (48px minimum)
  • Font sizes are readable (16px minimum for body text)
  • No intrusive interstitials or popups
  • Navigation is clear and intuitive
  • Search functionality is available
  • SSL certificate is valid and up to date

4. Off-Page SEO

  • Backlink profile has healthy domain authority
  • Toxic or spammy backlinks are disavowed
  • Competitor backlink gaps are identified
  • Google Business Profile is claimed and optimized (if local)
  • NAP (Name, Address, Phone) is consistent across web
  • Social media profiles are active and linked
  • Brand mentions are tracked

5. Monitoring & Analytics

  • Google Analytics is installed and tracking correctly
  • Google Search Console is configured
  • Goals/conversions are tracked
  • Keyword rankings are being monitored
  • Organic traffic trends are tracked monthly
  • Core Web Vitals are monitored in Search Console
  • Competition is tracked for key terms

Priority Scoring

For each failed item, assign a priority:

  • High — Directly impacts rankings or indexation. Fix within 1 week. (e.g., 404 errors, missing canonical tags, indexation issues)
  • Medium — Impacts user experience or competitive position. Fix within 1 month. (e.g., missing schema, slow pages, thin content)
  • Low — Incremental improvements. Fix within 3 months. (e.g., meta description optimization, alt text improvements)

Pro tip: Re-run this audit quarterly to track improvements and catch new issues before they impact rankings.
