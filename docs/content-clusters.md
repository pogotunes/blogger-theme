## Internal Linking Rules

### Pillar → Cluster Articles
Each pillar page must link to at least 3 supporting cluster articles.

### Cluster Articles → Pillar
Each cluster article must link back to its pillar page in the first 200 words.

### Cross-Cluster Links
| From Cluster | Link To |
|-------------|---------|
| Technical SEO | SEO Audit, Core Web Vitals |
| Local SEO | Google Business Profile, Reviews |
| AI SEO | Content Marketing, Technical SEO |
| Blogger SEO | Technical SEO, Speed Optimization |
| Link Building | Content Marketing, Digital PR |
| Ecommerce SEO | Technical SEO, Site Architecture |

### Global Links
Every page must link to:
- Contact page (`/p/contact.html`)
- Free SEO Audit page (`/p/free-seo-audit.html`)
- Related service pages (minimum 2 per page)
- GEO pages if relevant to location

### GEO Page Internal Linking Requirements
Every GEO page must:
1. Link to its country hub page (e.g., Mumbai → India)
2. Link to the main SEO Agency pillar page
3. Link to Local SEO Services pillar page
4. Link to 2-3 sibling city pages in the same country
5. Include CTA to Free SEO Audit

### Money Page Internal Linking
Each money page must:
1. Link to the main SEO Agency pillar page
2. Link to 1-2 related service pages
3. Link to 1-2 other money pages
4. Include CTA to Book Consultation or Free SEO Audit
